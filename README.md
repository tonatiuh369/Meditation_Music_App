# Meditation with Music - Web App

Web application with music to relax or meditate written in HTML/CSS/JavaScript. It consists of different audios that can be played with a visualizer at different time intervals. The audios are played with a video in the background.

# Meditación con Música - Aplicación web

Aplicación web con música para relajarse o meditar escrita en HTML/CSS/JavaScript. Consta de diferentes audios que se pueden reproducir con un visualizador a diferentes intervalos de tiempo. Los audios se reproducen con un video de fondo.
